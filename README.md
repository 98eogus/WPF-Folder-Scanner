# **WPF Folder Scanner**

WPF 기반 폴더 스캔 및 파일 확장자 통계 분석 도구

---

## **주요 기능**

- 파일 수 집계  
- 총 용량(bytes) 계산  
- 읽기 쉬운 용량(KB / MB / GB 자동 변환)  
- **TOP N 기능**: 가장 큰 용량을 가진 상위 N개의 확장자 조회  

---

## **개요**

MVVM 패턴을 적용하여 **View / ViewModel / Service** 계층을 분리하고  
**async/await 비동기 프로그래밍**을 통해 UI가 멈추지 않도록 구현하였습니다.

---

## **기술 스택**

- WPF (.NET 8)
- C# / XAML
- MVVM 패턴
- async / await 비동기 프로그래밍
