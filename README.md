📁 WPF Folder Scanner

WPF 기반 폴더 스캔 및 파일 확장자 통계 분석 도구

📄 파일 수
📦 총 용량(bytes)
🔍 읽기 쉬운 용량(KB, MB, GB 자동 변환)
⚡ TOP N 기능: 가장 큰 용량을 가진 상위 N 확장자 조회

MVVM 패턴을 적용하여 View, ViewModel, Service 계층을 분리하였으며
비동기 프로그래밍(async/await)을 통해 UI 멈춤 현상을 방지합니다.

🏗 기술 스택 (Tech Stack)
WPF (.NET 8)
C# / XAML
MVVM 패턴 적용
async / await 비동기 프로그래밍
